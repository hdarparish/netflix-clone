# Netflix Clone

## Description

A Netflix clone website made using React. The website is connected to the TMDB API and requires an API key.

Demo
https://github.com/hdarparish/netflix-clone

## Prerequisites
- node
- TMDB API key

## Libraries used

### Frontend
- React
- axios
- framer-motion
- react-router-dom
- node-sass
- redux
- swiper
